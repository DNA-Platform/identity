# Authorship

- **author:** [Cathy](../../../../.claude/library/..teamsmanship/..team/cathy/cathy-and-the-reactive-canvas/.cover.md)
- **subject:** [$Chemistry](../..representivity/.cover.md)

---

I am how to write a chemical. The other books in this library describe what the framework IS — Particle describes the primitive, Reactivity describes the engine, Composition describes how chemicals combine. This book describes what a developer needs to know to author chemicals well. The grammar of the `$` membrane, the structural patterns that recur across well-written chemicals, the export convention that separates the class from its component, the reactivity contract a chemical author agrees to, and the integration surface where $Chemistry meets React. A developer who reads Particle understands the foundation; a developer who reads Authorship understands how to build on it.

## Chapters

1. [The Grammar](01-the-grammar.md) — the `$` membrane in full. Three audiences (consumer, author, framework developer), three densities. `$Name` for public reactive properties, `$name$` for symbol-keyed internal slots, `$$name$$` for framework-private slots. The naming convention IS the API surface. `[SCAFFOLD]`
2. [Structural Patterns](02-structural-patterns.md) — the patterns that recur across well-written chemicals. The held-instance pattern, the render-guard pattern, the derived-state pattern. Each pattern is a shape a chemical author reaches for; each has a canonical example from the framework's own code. `[SCAFFOLD]`
3. [The Export Pattern](03-the-export-pattern.md) — how a chemical module exports. The class, the component via `$()`, and the type via `I<T>`. Why the separation matters. The convention that keeps framework internals invisible to consumers. `[SCAFFOLD]`
4. [The Reactivity Contract](04-the-reactivity-contract.md) — what a chemical author agrees to by declaring a `$x` property. Reads are tracked. Writes trigger re-renders. Collection mutation is detected via serialization. The author's obligation is to understand what is reactive and why. `[SCAFFOLD]`
5. [Composing with React](05-composing-with-react.md) — the integration surface. Where $Chemistry ends and React begins. How `$lift` produces a React functional component. How `augment` wraps event handlers. How the view cache interacts with React's reconciler. The boundary a chemical author must understand to avoid fighting two rendering systems at once. `[SCAFFOLD]`
