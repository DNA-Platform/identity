The team's knowledge of building $Chemistry, the reactive framework about representation. $Chemistry models how reactive state represents something beyond itself — how a particle carries a value, how a chemical composes particles into a view, how an atom bonds chemicals into shared substrate. Representivity catalogues what the team learned building that representational framework: the decisions, the corrections, the patterns that emerged from forty sprints of applied work.

This branch library lives at `library/chemistry/.lib/` beside the framework source. It follows the same [Bookkeeping](../../../../.claude/library/bookkeeping/.cover.md) conventions as the main library — covers, chapters, the dot type system. Links point INTO the identity library for protocols, conventions, and autobiographies. The identity does NOT link into this branch, following the [one-way link convention](../../../../.claude/library/library-tree/01-branches.md#the-one-way-link-convention).
