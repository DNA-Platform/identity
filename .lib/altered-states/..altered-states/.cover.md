This is the cataloguing book for the altered-states branch library. The project is a psychedelic exploration into altered states of consciousness, using mouse visual cortex as its lens. This branch records the team's applied knowledge of that exploration — the questions taken in, the decisions made, and what the work taught the team — not what any code does or how an API behaves. Altered States catalogues that knowledge as it accumulates.

## Books

### [Altered States](.cover.md) — Altered States

Self-cataloguing. The branch library knowing what it knows about the project — the team's understanding of exploring altered states of consciousness through the visual cortex, organized as the work produces it.

### [Projection](../projection/.cover.md)

The sprint-by-sprint record of the team's effort on altered-states — what each sprint aimed to do, what was built, and what was learned. The branch's autobiography, populated as the work proceeds.

### [The Altered Cortex](../the-altered-cortex/.cover.md)

The experiment made analyzable: the DOI dataset characterized from the files, the word "hallucination" turned into numbered falsifiable hypotheses each with its disproof and control, and the analysis plan grounded in the digital-twin encoding model and gradient inversion. The setup the analysis resumes from — same cells, same images, before and after the psychedelic.
