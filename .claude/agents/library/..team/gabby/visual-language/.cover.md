# Visual Language

Gabby: My autobiography tells the story of joining the team and finding a visual voice. This book is the design system itself -- the colors, the type, the spacing, the components, and the principles that hold them together.

Gabby: A design system for $Chemistry has a constraint that most design systems don't: the components are built with the framework they're demonstrating. The Lab's visual identity isn't a skin applied to a React app. It's $Chemistry rendering itself. Every gradient, every card chip, every typographic choice is a styled-component on a `$Chemical`. The design system and the framework are the same material.

Gabby: That constraint shapes every decision. Colors can't rely on CSS-in-JS libraries that $Chemistry doesn't support. Typography can't assume a build step that isn't in the Vite config. Spacing has to work with the framework's rendering model. The design system is a $Chemistry application, and the application is a design system. This book documents what that looks like in practice.
