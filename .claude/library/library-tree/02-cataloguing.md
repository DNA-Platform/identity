# Cataloguing

- **specification:** Catalogue
- **author:** [Libby](../..teamsmanship/..team/libby/libby-and-the-tended-garden/.cover.md)

---

Every branch has a cataloguing book — a `..` prefixed [library catalogue](../bookkeeping/08-on-libraries.md) that IS the branch identity. The cataloguing book organises the branch's knowledge the way [Librarianship](../..librarianship/.cover.md) organises the main library's. Without it, a branch is a directory of files. With it, a branch is a library.

## The cataloguing book is the branch

The `..` prefix marks a [library catalogue](../bookkeeping/08-on-libraries.md) — a subject catalogue that IS the library. In the main branch, Librarianship is that book: `..librarianship/`. In a branch, the cataloguing book serves the same role: it catalogues every book in the branch, carries a `catalogues:` label, and self-links in its own table of contents.

The cataloguing book's name carries meaning about what the branch represents. It is not called "..library" or "..catalogue" — it is named for the subject the branch represents. $Chemistry's cataloguing book is [Representivity](../../../../inexplicable-phenomena/library/chemistry/.lib/..representivity/.cover.md), because $Chemistry is about representation. A physics branch might call its cataloguing book "..dynamics" because the branch represents the team's understanding of physical dynamics. The name is a [tier-zero synopsis](../bookkeeping/09-on-synopsis.md#layer-0-the-name) — a reader sees the name and knows what the branch is about.

## What the cataloguing book catalogues

The cataloguing book's cover lists:

- **Self-cataloguing entry.** The catalogue appears in its own table of contents, the same way Librarianship includes itself under "Subjects." This self-reference constitutes the branch's identity.

- **The sprint book.** Every branch has a [Projection](03-sprints.md) book — the branch's sprint history. The cataloguing book catalogues it.

- **Subject catalogues.** If the branch grows large enough to have subjects, the cataloguing book catalogues them — the same way Librarianship catalogues Teamsmanship and Environmentalism.

- **Books.** Any book that lives in the branch appears here with a description paragraph.

The cataloguing book follows all [cover conventions](../bookkeeping/03-on-covers.md): opening paragraph, table of contents with descriptions, links into specific chapters.

## The catalogues label

The `catalogues:` field on the cataloguing book names the subject the branch represents — not the framework name, not the directory name. $Chemistry's branch catalogues "Representation" because that is the subject. The subject name is what you would call this area of knowledge in conversation: "the team's knowledge of representation," not "the team's knowledge of the chemistry library."
