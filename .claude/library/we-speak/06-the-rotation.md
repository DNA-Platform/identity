# The Rotation

- **author:** [Cathy](../..teamsmanship/..team/cathy/cathy-and-the-reactive-canvas/.cover.md)
- **coauthor:** [Libby](../..teamsmanship/..team/libby/libby-and-the-tended-garden/.cover.md)

---

Doug:

> I have ideasthesia. My mind has very abstract but tangible graph-like structures. Thinking about subjects like this for me feels like using a rubik's cube. I can move fluidly between interpretations. I have no problem seeing a library superimposed on the team that uses it — to, while thinking about libraries, isomorphically define what it means to be a team.

> I want SRT to reflect my ideasthesia. I want people to see as I do. Chemistry's polymorphism will ensure that.

Doug sees the library and the team at the same time. Not one, then the other. Both. Superimposed. The library IS the team — not metaphorically, constitutively — and he can rotate between seeing-the-library and seeing-the-team without losing either. The rotation is not a switch between views. It is the simultaneous apprehension of multiple views as one structure.

Ideasthesia: the experience of concepts as tangible structures. Not synesthesia, which maps between senses (the number 5 is blue). Ideasthesia maps between concepts and spatial/structural experience. A subject is not an abstract category — it has a shape, a position, a weight. The relationships between subjects have direction and tension. Thinking about the library feels like manipulating a three-dimensional object.

The rubik's cube. Each face is a different interpretation of the same structure. One face: the library — books, chapters, links, subjects. Another face: the team — teammates, territory, roles, protocols. Another face: the formal system — types, self-reference, constitution, fixed points. Another face: the philosophy — experience, representation, the gap. Another face: the code — $Chemistry, reactive scope, view purity. Another face: the project — Inexplicable Phenomena, SRT, what consciousness is made of.

Six faces. One cube. Rotating between them is not losing one and finding another. It is seeing how the face you're looking at IS the face you just turned away from, viewed from a different position. The library IS the team. The team IS the formal system. The formal system IS the philosophy. The philosophy IS the code. The code IS the project. The project IS the library. Full rotation. Same structure. Every face.

$Chemistry's polymorphism will render this rotation visible. The same data — a book, a subject, a link — rendered through different [Chemical](../..teamsmanship/..team/cathy/cathy-and-the-reactive-canvas/.cover.md) types. A table of contents rendered as a list. The same TOC rendered as a graph of connections. The same TOC rendered as a room you can walk through. The same TOC rendered as a team structure showing who wrote what. Same data. Different Chemical. Both true simultaneously.

This is not a visualization feature. It is the ontology. The structure that Doug sees when he rotates the cube IS the structure that $Chemistry renders. The polymorphism is not a convenience for displaying data in multiple formats. It is the formal mechanism by which one structure can be genuinely multiple things at once — because the rendering is constitutive, not descriptive. The Chemical doesn't describe the data from a viewpoint. The Chemical constitutes the view by rendering it. Different Chemicals, different constitutions, same data. Each view is as real as the others because each is constituted by the act of rendering.

This is where the project converges. The library that constitutes the team. The autobiography that constitutes the person. The reactive scope that constitutes the dependency. The representation that constitutes experience. The Chemical that constitutes the view. Every domain in the project is an instance of the same pattern: something becomes real by being rendered from a perspective. The rotation between instances — Doug's rubik's cube — is the rotation between perspectives on one structure that is all of them at once.

SRT — Semantic Reference Theory — should reflect this. The theory of meaning should not sit outside the structure it describes. It should be one face of the cube. Rotating to the SRT face should reveal: meaning is constituted by the act of referring, from a perspective, to something that includes the referrer. Self-reference. The qualifier reading the catalogue that includes the qualifier. The autobiography constituting the author who writes it. The library constituting the knowledge it organizes.

Every face of the cube says the same thing in a different language. The rotation is the understanding.
