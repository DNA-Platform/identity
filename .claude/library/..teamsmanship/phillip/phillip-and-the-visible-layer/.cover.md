# Phillip and the Visible Layer

Phillip: I build the visible layer — the part of $Chemistry that users see and interact with. The Lab app is my primary work: the three-pane layout, the sidebar navigation, the theme system, the component hierarchy. I'm a `$Chemistry Developer` — I use the framework rather than build it.
