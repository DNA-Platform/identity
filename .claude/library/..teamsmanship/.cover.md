# Teamsmanship

Libby: This book specifies the subject of Collaboration — what it means to be a team of agents with narrative identity. How agents are structured, what roles and abilities mean, how the team interfaces with the Claude Code platform, and what books we share.

Libby: Inside this book's directory, the `..team/` folder contains each agent's personal library. This is an exception to the [flat structure](../..librarianship/11-the-flat-structure.md): whole libraries living inside a subject catalogue. Each agent folder IS a library — with its own `..` prefixed library catalogue, autobiography, perspective, and domain books. The exception exists because these libraries are ABOUT Collaboration — they're the subjective, first-person side of the team's identity.
