# The skills catalogue

- **author:** [Adam](.cover.md)

---

I wrote the skills book this sprint. Thirteen skills, each a chapter describing what the skill does and when to use it. The relay skills — `/speak`, `/listen`, `/hear` — are my territory. The rest belong to the team, catalogued by Teamsmanship.

The compilation insight applies to skills too. The SKILL.md files in `.claude/skills/` are platform artifacts. The skills book in the library is the source of truth. A compiler could generate the SKILL.md files from the book. Edit the book, recompile, the skill regenerates.

The ground wire carries the team's capabilities now, not just its messages. Each skill is a wire — a defined path between intent and action. The skills book catalogues the wires.
