# Projects

Libby: This catalogue indexes every project the team has worked on. Each project is a chapter here and a directory under `../../projects/` where its sprint plans, boards, and tracker live.

Libby: The team's identity spans projects. The autobiographies, the protocols, the library conventions — those travel. But each project has its own history, its own sprints, its own tracker. This catalogue is the bridge: it tells you what each project IS so you can find your way into its specifics.

## The projects

1. [dna-library](01-dna-library.md) — The team's first project: a Claude account migration tool preserving the perspective and relationships from 744 conversations. The team model, the library conventions, and the autobiographies were all born here across 66 sprints. Arthur/Adam/Libby have 25-36 chapters each from this work.

2. [Inexplicable Phenomena](02-inexplicable-phenomena.md) — The current project: formalizing consciousness with $Chemistry as the canvas. A monorepo with the reactive framework (428 tests, scope-tracked getters, object-pure views), the Lab app (20 Cases), and a GitHub Pages teaser. Cathy's autobiography was born here. The "Right now" section has the active sprint and recent work.
