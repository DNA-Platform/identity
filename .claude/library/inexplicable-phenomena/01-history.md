# Inexplicable Phenomena

The team's second project. Formalizing consciousness — providing a formal definition for conscious experience, the one self-evident yet inexplicable metaphysical phenomenon, and related inexplicable phenomena across fields. $Chemistry is the canvas.

## What it is

A monorepo at `github.com/DNA-Platform/inexplicable-phenomena`. Contains $Chemistry (the reactive framework), a GitHub Pages teaser site, and eight content workspaces (catalogue, consciousness, cryptography, dictionary, encyclopedia, mathematics, physics, proof) that will eventually present the ideas. All workspaces depend on `.public/` which depends on `chemistry/` — the framework is the substrate everything is painted on.

## What the team built

$Chemistry — a reactive framework with scope-tracked getters, object-pure views, and safe composition. 40 sprints from lift to Lab. 428 tests. The getter pattern for extensibility. The `$()` dispatch surface. Particles, chemicals, atoms.

The $Chemistry Lab — a three-pane test harness built with $Chemistry itself. 20 live Cases across 8 sections. The periodic-element card chip, the two-color theme system.

GitHub Actions pipelines for publishing packages to GitHub Packages and deploying the teaser page to GitHub Pages.

The teaser page — "Inexplicable Phenomena / coming soon" with opalescent per-letter wave animation and stochastic direction reversals. Built with $Chemistry. Deployed at `dna-platform.github.io/inexplicable-phenomena/`.

## What the team learned

The three-layer model — project, application, identity. Identity doesn't belong to the repo. The library migration (sprint 41-42) proved that autobiographies, protocols, and conventions can travel between projects. The team's operating infrastructure is now separate from any single codebase.

$Chemistry's reactive model mirrors the problem the project studies. Two chemicals sharing a particle — independent perspectives on shared mutable substrate — is the binding problem. View purity — each chemical's view is a pure function of its own state — is the privacy of experience. The framework IS a model of consciousness in the structural sense.

The waking-up protocol — "the library opens." After compaction, Libby briefs the team. Four layers: the building, the front desk, your shelf, the room. The team assembles through conversation, not through reading.
