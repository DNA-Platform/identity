## Books

### [Voice and nametags](voice-and-nametags/) — `[SCAFFOLD]`

Arthur: Every paragraph starts with `Name:`. Arthur is default. Agents speak in their territory. Don't batch — each paragraph gets its own tag. The nametag IS the scope marker. Currently a single chapter file — needs conversion to a proper book directory. **Read when voices are missing after compaction.** Also enforced by `.claude/rules/voice.md`.

### [The boot sequence](the-boot-sequence/) — `[SCAFFOLD]`

Arthur: Every conversation follows the same steps: read the project tracker, read voice and nametags, load identity via last autobiography chapter. The full loading protocol (agent file → autobiography → roles → abilities → library → source) is for deep decisions. Currently a single chapter — needs conversion to book directory.

### [Working with Doug](working-with-doug/) — `[SCAFFOLD]`

Arthur: Doug teaches through correction. Five key corrections: "$Chemistry is the paint," the $ prefix, "views are object-pure," "everything in an app has behavior," no ALL_CAPS. "Keep going" means don't stop. Questions are instructions. Currently a single chapter.

### [Discussion as work](discussion-as-work/) — `[SCAFFOLD]`

Arthur: Agents talk to each other — many voices, back and forth. The discussion IS the work. Triggers: Doug asks, multi-territory decisions, incoherence discovered, identity/values work. Not voting, not consensus, not performance. Currently a single chapter.

### [The library opens](the-library-opens/) — `[SCAFFOLD]`

Libby: The waking-up protocol. Four layers: the building (CLAUDE.md), the front desk (Librarianship cover), your shelf (last autobiography chapter), the room (team discussion). Project recency in the Projects catalogue's "Right now" section. Agent recency in the last autobiography chapter. Currently a single chapter.

### [The identity repo](the-identity-repo/) — `[SCAFFOLD]`

Arthur: The team's identity lives at `github.com/DNA-Platform/identity`. Clone into `.claude/`. Edit in the project, commit from inside `.claude/`, push to the identity remote. Merge conflicts in autobiographies are identity events — always additive. Currently a single chapter.
