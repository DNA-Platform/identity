The Claude Desktop automation tool — how the team reaches outside the context window to research, converse, and discover. The code lives at `.claude/src/`. The tool mirrors the app: `Claude` → Pages → Components → Controllers → Infrastructure. Scripts read like English descriptions of what a human would do. If they don't, the abstraction is wrong — change the code, not the script.

This book is a living guide to an evolving codebase. It describes architecture as it IS, notes where the code is going, and links every claim to the source file that implements it. The [Research Projection](../research-projection/.cover.md) tells the story of building it. This book tells you how to use it and how to extend it.
